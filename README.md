# DistilBERT fine-tuning for text classification

## Getting Started
### Pre-requisites
- Python 3.11 or later
- Poetry
  ```
    curl -sSL https://install.python-poetry.org | python3 -
  ```
### Installation
1. Clone the repository
    ```
    git clone
    ```
2. Create a virtual environment
    ```
    poetry shell
    ```
3. Install project dependencies
    ```
    poetry install
    ```
